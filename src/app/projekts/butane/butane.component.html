Butane

Butane is an open source secret vault. I've used Angular on client side and AWS Lambda with various languages on server side. (currently Golang and Node.js)
Butane utilizes different OAuth providers for authentication, Current providers are G o o g l e and Discord.

I primarily made this project to learn various AWS services such as CloudWatch, Lambda, DynamoDB, API Gateway, DDB Event Streams, etc and to
learn setting up CI/CD pipeline with OIDC authentication using GitHub Actions.

I've also wanted to learn Terraform to manage AWS resources and Infrastructure-as-Code so I've used Terraform to manage
AWS resources and GitHub Actions to manage CI/CD pipeline and deploy it to AWS.
