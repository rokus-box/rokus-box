redi-rs

redi-rs is a Lightweight Redis client with connection pooling written in Rust and 100% memory safe

I created this library to learn networking with Rust, working with threads,
streams and how to make these thread-safe. I was wondering what could I do with raw
TCP connections and I thought that a Redis client with connection pooling would be a good
idea to learn all these things. :)

In the following example, this client establishes a connection pool of 10 connections
to a Redis server running on localhost:6379 and then executes
commands by using the connection pool and gives back the connection to the
pool. If the pool does not have any available connections, the client will
create a new connection and dismiss it after the command is executed.
